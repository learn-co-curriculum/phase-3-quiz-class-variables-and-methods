# OO Syntax

What is the syntax for a class variable?

## Correct

```
@@variable_name
```

> Correct! We use `@@` to declare class variables.

## Incorrect

```
@variable_name
```

> This is how we declare instance variables.

## Incorrect

```
$variable_name
```

> This is how we declare global variables.

## Incorrect

```
VARIABLE_NAME
```

> This is how we declare a class constant.

## Incorrect

I don't know
