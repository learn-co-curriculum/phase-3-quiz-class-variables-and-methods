# OO Syntax

Classes themselves are objects.

## Correct

True

> behaviors, then an entire class itself qualifies as an object. Our custom Ruby
> classes are actually instances of the
> [`Class` class](https://ruby-doc.org/core-2.5.3/Class.html).
